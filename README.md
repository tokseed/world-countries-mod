# Mod: Столицы и страны мира (Unlimited Hints)

**❗ ВАЖНО: Если вы скачали файл с расширением `.zip`, обязательно переименуйте его обратно в `.apks` после загрузки для корректной работы в SAI.**

Репозиторий содержит модифицированную сборку мобильного приложения «Столицы и страны мира» для платформы Android. Основная цель модификации — разблокировка игровых ресурсов (подсказок) для обеспечения неограниченного игрового процесса.

## Основные изменения
* **Patch Hints:** Значение доступных подсказок в игровом профиле установлено на уровне **99 999 997**.
* **Build Optimization:** Произведено выравнивание данных (zipalign) для оптимизации потребления оперативной памяти и ускорения запуска приложения.

## Состав репозитория
* `game_mod.apks` — итоговый пакет формата Split APK, готовый к установке.
* `modified-aligned-debugSigned.apk` — модифицированный базовый программный модуль (base.apk).

## Инструкция по установке

В связи с тем, что приложение использует архитектуру **Split APKs**, стандартная установка через системный менеджер файлов может привести к ошибке. Рекомендуется использовать один из следующих методов.

### Метод 1: Использование SAI (Split APKs Installer)
1. Удалите оригинальную версию приложения.
2. Запустите **SAI** и перейдите в раздел **Настройки**.
3. В меню «Установка» активируйте параметр **«Подписывать перед установкой» (Sign before install)**. Это необходимо для устранения конфликта подписей между модифицированным базовым модулем и оригинальными сплитами.
4. Выберите файл `game_mod.apks` и инициируйте процесс инсталляции.

### Метод 2: Использование MT Manager
1. Откройте `game_mod.apks` через интерфейс **MT Manager** (команда «Просмотр»).
2. Выберите файл `base.apk` внутри архива.
3. Запустите установку и подтвердите запрос на автоматическую переподпись пакета (Sign).

## Техническая спецификация
* **Architecture:** ARM64 / ARMv7
* **Signature:** Debug Key
* **Tools used:** `apktool` (Smali editing), `uber-apk-signer`, `bundletool`.
